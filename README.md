# API de Usuários - ASP.NET Web API

##  Tecnologias
- C#
- ASP.NET Web API
- PostgreSQL
- Entity Framework

##  Funcionalidades
- Criar personagem
- Listar personagens
- Atualizar personagem
- Deletar personagem

##  Endpoints
GET /api/personagens  
POST /api/personagens  
PUT /api/personagens/{id}  
DELETE /api/personagens/{id}

##  Como rodar
1. Clone o repositório
2. Configure o banco PostgreSQL
3. Execute com `dotnet run`

##  Testes
A API foi testada utilizando Swagger com suporte completo a CRUD.
